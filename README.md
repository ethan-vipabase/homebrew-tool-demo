## For demo purpose

This repo is to demo how to distribute a custom tool via Homebrew


## How to install it

```sh

# step 1: Download the tool's binary into Homebrew' tap

brew tap ethan-vipabase/homebrew-tool-demo

# step 2: Install the tool

brew install ethan-vipabase/homebrew-tool-demo


```

or run `install.sh`

